# SolidFind — Web UI kit

A plausible product surface (marketing + listing browse) using only the brand language present in the Figma file. **Not** a recreation of a shipping product — there is no codebase or product Figma in the source materials.

## Files

| File | What it is |
|---|---|
| `index.html` | Click-thru: marketing hero → search/listing view → pro detail. |
| `Header.jsx`     | Top nav with wordmark + search + account chip |
| `Hero.jsx`       | Full-bleed gradient hero with the brand voice |
| `SearchBar.jsx`  | The primary search affordance |
| `Filters.jsx`    | Category + location filter row |
| `ProCard.jsx`    | Default listing card |
| `ProDetail.jsx`  | A pro's profile / detail page |
| `Footer.jsx`     | Quiet footer with locale + IG link |

All visuals are driven by `../../colors_and_type.css`. Assets resolve to `../../assets/*`.
